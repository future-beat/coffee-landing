  <name>Task 2: Configure design tokens in globals.css</name>
  <files>coffee-landing/app/globals.css</files>
  <action>
Replace the contents of globals.css with Tailwind v4 configuration including brand design tokens:

```css
@import "tailwindcss";

@theme {
  /* Brand Colors */
  --color-brand-cream: oklch(0.95 0.03 90);
  --color-brand-espresso: oklch(0.25 0.05 30);

  /* Typography */
  --font-sans: 'Inter', sans-serif;
  --font-display: 'Playfair Display', serif;

  /* Spacing */
  --spacing-section: 6rem;
}
```

These tokens will be available as Tailwind utilities (e.g., `bg-brand-cream`, `text-brand-espresso`, `font-display`).
  </action>
